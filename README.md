# 🐈 Cute Cat with Butterfly Animation

A fun HTML and CSS project featuring a cute black cat inside a circular frame with a rotating border and a flapping butterfly animation.

## ✨ Features

- 🐱 CSS-drawn cat with ears, eyes, whiskers, body, and tail
- 🦋 Animated butterfly with wing flapping effect
- 🔄 Rotating dotted outer circle
- 🎨 Circular frame with decorative dashed border
- 🌄 Scenic background image
- 📐 Created entirely using HTML and CSS
- 💫 Uses CSS animations, transforms, positioning, and overflow properties

## 🛠 Technologies Used

- HTML5
- CSS3
  
## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/cute-cat-animation.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## 📚 Concepts Practiced

- Absolute and relative positioning
- Parent-child relationships
- Overflow hidden
- z-index layering
- CSS transforms
- Scale and rotation
- CSS keyframe animations
- Border radius and custom shapes
- Transform origin
- Animation timing functions

## 🎯 Project Goal

This project was created to practice CSS art and animation techniques by building a cute cat illustration and adding interactive elements such as a rotating frame and a butterfly with flapping wings.

Made with ❤️ using HTML and CSS
